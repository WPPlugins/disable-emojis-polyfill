Disables WordPress emojis backward compatibility.

== Description ==

Disable Emojis Polyfill  plugin removes WordPress emojis backward compatibility for older browsers.

It removes both styles and scripts from front-end and back-end pages, including the content and comment feeds and emails.

== Installation ==

This section describes how to install the plugin and get it working.

= Normal plugin =

1. Upload the plugin files to the `/wp-content/plugins/disable-emojis-polyfill` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress

= Must-use plugin =

1. Upload the `disable-emojis-polyfill.php` file into `/wp-content/mu-plugins` folder.
